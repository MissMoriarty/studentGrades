Student Grades - Sistema de calificaciones básico

Este proyecto es una aplicación de consola en **C# (.NET)** diseñada para automatizar el cálculo de calificaciones de estudiantes. Procesa puntuaciones de exámenes y tareas de crédito extra para generar un reporte final con promedios y letras de calificación (A-F).

El sistema toma un conjunto de puntuaciones predefinidas para varios estudiantes y realiza los siguientes cálculos:

1. Separa las calificaciones en **Exámenes** y **Crédito extra**.
2. Calcula el promedio ponderado, donde el crédito extra suma puntos adicionales al puntaje total de los exámenes.
3. Asigna una **Letra de calificación** (Grade Letter) basada en el promedio final.
4. Imprime un reporte tabulado en la consola.


---

> **Nota:** Este proyecto es un ejercicio práctico sobre el manejo de `Arrays`, bucles `foreach` y lógica condicional en C#.
